# Employee Management System

Employees **check in/out** and log breaks; admins **create accounts**, manage **projects** (create, assign, update), and view **monthly time logs** per employee.

**Stack:** Python · Flask · SQLite · Jinja2 · Werkzeug (password hashing) · HTML/CSS

**Run:** `pip install flask` → `python app.py` → open http://127.0.0.1:5000
